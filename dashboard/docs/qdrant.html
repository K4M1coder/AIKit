Overview

Qdrant is a high-performance vector database designed for storing, searching, and managing vector embeddings. It's perfect for building AI applications that require semantic search and similarity matching.

Quick Start

  1. Access Qdrant at localhost:6333
  2. View the web dashboard to manage collections
  3. Create your first collection via API or dashboard
  4. Start indexing vectors and performing searches

Configuration

Environment Variable Value Description
QDRANT__SERVICE__HTTP_PORT 6333 HTTP API port
QDRANT__SERVICE__GRPC_PORT 6334 gRPC API port
QDRANT__LOG_LEVEL INFO Logging level
