Overview

Elasticsearch is a distributed, RESTful search and analytics engine capable of addressing a growing number of use cases. It provides real-time search and analytics for all types of data.

Quick Start

  1. Access Elasticsearch at localhost:9200
  2. Check cluster health: GET /_cluster/health
  3. Create an index and start indexing documents
  4. Use the REST API to search and analyze data

Configuration

Environment Variable Value Description
discovery.type single-node Single node cluster configuration
xpack.security.enabled false Disable security for development
ES_JAVA_OPTS -Xms512m -Xmx512m JVM heap size configuration
