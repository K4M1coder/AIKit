Overview

CrewAI Studio is a platform for orchestrating AI team agents that work together to accomplish complex tasks. It enables you to create, manage, and deploy collaborative AI agents with different roles and capabilities.

Quick Start

  1. Access CrewAI Studio at localhost:8501
  2. Create your first crew with multiple AI agents
  3. Define roles, goals, and backstories for each agent
  4. Set up tasks and workflows for agent collaboration

Configuration

Environment Variable Value Description
STREAMLIT_PORT 8501 Port for CrewAI Studio web interface
OPENAI_API_KEY your-key OpenAI API key for agent LLM access
CREWAI_STORAGE_DIR /app/storage Directory for storing crew configurations
