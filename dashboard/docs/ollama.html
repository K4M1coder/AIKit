Overview

Ollama is a lightweight and extensible framework for running large language models locally. It provides OpenAI-compatible API endpoints and supports various popular LLM models like Llama 2, Code Llama, and many others.

Quick Start

  1. Access Ollama API at localhost:11434
  2. Pull a model: ollama pull llama2
  3. Start chatting: ollama run llama2
  4. Use the OpenAI-compatible API endpoint for integrations

Configuration

Environment Variable Value Description
OLLAMA_HOST 0.0.0.0:11434 Host and port for Ollama server
OLLAMA_MODELS /root/.ollama/models Directory for storing models
OLLAMA_NUM_PARALLEL 1 Number of parallel requests
