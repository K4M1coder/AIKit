Overview

n8n is a workflow automation platform that lets you connect different services and build automated workflows using a visual editor.

Quick Start

  1. Access n8n at localhost:5678
  2. Create an account or login with default credentials
  3. Start creating workflows using the visual editor

Configuration

Environment Variable Value Description
DB_TYPE postgresdb Database type for n8n
DB_POSTGRESDB_HOST postgres PostgreSQL database host
N8N_DIAGNOSTICS_ENABLED false Disable sending diagnostic data
