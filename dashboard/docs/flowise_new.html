Flowise Documentation

Overview

Flowise is a drag & drop UI to build your customized LLM flow. It allows you to build conversational AI applications using a visual interface, making it easy to create complex AI workflows without coding.

Key Benefits:
  • Visual drag-and-drop interface
  • No-code LLM application development
  • Support for multiple LLM providers
  • Built-in vector database integration

Quick Start

  1. Access Flowise through the AIKit dashboard
  2. Create a new chatflow using the visual editor
  3. Drag and connect nodes to build your workflow
  4. Configure your LLM and other components
  5. Test and deploy your AI application

Configuration

Flowise configuration involves several key components:

  • LLM Models: Configure OpenAI, Anthropic, or local models
  • Vector Stores: Set up Pinecone, Chroma, or other vector databases
  • Memory: Configure conversation memory and context
  • Tools: Add custom tools and integrations

Features

Drag & Drop Interface

Build AI workflows visually with an intuitive drag-and-drop interface.

Multiple LLM Support

Integrate with various LLM providers including OpenAI, Anthropic, and local models.

Vector Database Integration

Built-in support for popular vector databases for RAG applications.

Conversation Memory

Maintain context across conversations with built-in memory management.
