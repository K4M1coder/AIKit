Overview

Flowise is a visual LangChain builder that lets you create AI workflows and chatflows using a drag-and-drop interface. It's perfect for building complex AI applications without writing code.

Quick Start

  1. Access Flowise at localhost:3002
  2. Explore the template gallery for pre-built workflows
  3. Create your first chatflow using the visual builder
  4. Connect to your preferred LLM and vector database

Configuration

Environment Variable Value Description
PORT 3002 Port for Flowise web interface
FLOWISE_USERNAME admin Default username for authentication
FLOWISE_PASSWORD 1234 Default password for authentication
