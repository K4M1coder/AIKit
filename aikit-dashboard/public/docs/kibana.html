Overview

Kibana is a powerful visualization and exploration tool for Elasticsearch data. Use it to create dashboards, discover trends, and monitor your systems.

Quick Start

  1. Access Kibana at localhost:5601
  2. Connect to your Elasticsearch instance
  3. Create an index pattern for your data
  4. Use Discover and Dashboards to explore and visualize

Configuration

Setting Value Description
server.port 5601 Port for Kibana web UI
elasticsearch.hosts http://elasticsearch:9200 Elasticsearch endpoint
server.publicBaseUrl http://localhost:5601 Public Base URL
