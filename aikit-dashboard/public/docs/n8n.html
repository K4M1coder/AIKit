Overview

n8n is a powerful workflow automation tool that integrates with hundreds of services. In AIKit, it orchestrates agents, data pipelines, and Slack integrations.

Quick Start

  1. Access n8n at localhost:5678
  2. Sign in and open the AI agent workflows
  3. Configure credentials (Slack, Google, Postgres) as needed
  4. Run the workflow or trigger via webhook

Configuration

Variable Example Description
N8N_PORT 5678 Port for n8n UI
N8N_HOST n8n Hostname used by other containers
N8N_PROTOCOL http Protocol for external URL
N8N_EDITOR_BASE_URL http://localhost:5678 Public editor URL
N8N_ENCRYPTION_KEY ... Key to encrypt credentials
