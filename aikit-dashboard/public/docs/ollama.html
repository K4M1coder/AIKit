Overview

Ollama runs large language models locally with a simple API. In AIKit, Ollama powers LLM features for agents and tools.

Quick Start

  1. Ensure Ollama is running locally at http://localhost:11434
  2. Pull a model, e.g., ollama pull llama3
  3. Send a prompt via API: POST to /api/generate with model and prompt

Configuration

Setting Default Description
OLLAMA_HOST 0.0.0.0 Bind address
OLLAMA_PORT 11434 API port
OLLAMA_MODELS ~/ollama/models Models storage directory
