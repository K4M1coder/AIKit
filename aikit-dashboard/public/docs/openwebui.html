Overview

OpenWebUI is a web interface for interacting with local LLMs powered by Ollama. It provides a clean chat experience with prompt history and model controls.

Quick Start

  1. Launch OpenWebUI at localhost:8080
  2. Ensure Ollama is running with at least one model pulled
  3. Open a chat, select a model, and start prompting

Configuration

Setting Default Description
OPENWEBUI_PORT 8080 Web UI port
OLLAMA_API_BASE http://ollama:11434 Backend API base URL
