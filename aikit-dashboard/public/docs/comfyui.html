Overview

ComfyUI is a powerful and modular stable diffusion GUI and backend. It's designed to be a node-based interface for creating complex image generation and manipulation workflows.

Quick Start

  1. Access ComfyUI at localhost:8188
  2. Load a checkpoint model or download one from the queue manager
  3. Create a workflow by connecting nodes in the interface
  4. Configure your prompt and generation settings
  5. Queue the prompt to generate images

Configuration

Environment Variable Value Description
COMFYUI_PORT 8188 Web interface port
COMFYUI_LISTEN 0.0.0.0 Listen address for external access
CUDA_VISIBLE_DEVICES 0 GPU device selection
